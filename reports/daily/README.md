# 每日 PDF

本目录保存面向快速阅读的每日 PDF，路径为：

```text
reports/daily/YYYY/MM/软件工程洞察_YYYY-MM-DD.pdf
```

从 2026-07-19 起使用上述命名。更早已经发布的 `YYYY-MM-DD.pdf` 作为历史兼容文件保留，不重命名，以免破坏既有链接。

每日 PDF 由当天两个专题日报“今日精选洞察”、有效信号记录和已审阅图示派生。首页提供委员会署名、日期、关键词和可跳转的洞察汇总；正文展示中文编辑标题、原始标题、来源类型、分项观点总结、关键词、可选技术图示、结构化工程方案、领域启示和可点击原文/佐证来源。洞察数量和 PDF 总页数不设上限，零洞察时生成一页“今日无重要更新”。

专题日报与信号记录仍是事实、证据状态和更新历史的正式知识载体；PDF 不重新研究或补写结论，也不显示证据标签、来源优先级、关联信号和尚待验证。

日度 PDF 不展开跨洞察趋势。趋势、方向变化、证据升级和实践启示由周报及后续周报 PDF 承载。
