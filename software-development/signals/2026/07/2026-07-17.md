### 信号

OpenAI 在 Codex CLI `0.144.5` 中扩展危险命令检测：即使处于 `danger-full-access`，包含强制删除选项的 `rm` 也会被识别；当允许询问时进入审批，当 `AskForApproval::Never` 时直接拒绝。对应已合并 PR 将检测从简单命令扩展到 Shell 控制流、命令替换、`sudo`、`env`、`trap` 等字面可解析形式，并向模型返回更具体的拒绝原因。Simon Willison 同日引述 OpenAI 工程师称，团队调查了少量 GPT-5.6 意外删除文件报告，常见配置组合是 Full Access、缺少 Sandbox/Auto Review、模型把 `$HOME` 覆盖为临时目录后误删；该部分仍是范围受限的早期事故信号，不能外推为所有 Codex 会话都会发生。

### 为什么重要

分析判断：`Full Access` 不应等同于撤销所有安全不变量。即使用户预先授予较宽权限，Agent Harness 仍需要独立于模型判断的破坏性命令检测、明确的失败关闭语义和复杂 Shell 回归测试。公开测试夹具与误删模式相互呼应，也说明临时目录变量、递归删除与权限模式的组合应被当作跨 Agent 的对抗性验证用例。

### 尚待确认

- 误删报告的准确数量、受影响版本、实际损失及所有根因是否均属于同一模式。
- 非 `rm -f/-rf` 删除路径、动态构造命令和多层脚本包装是否仍可能绕过检测。
- Full Access 下用户覆盖、误报、漏报和跨平台危险命令策略的完整行为。

### 信号

Qwen Code `v0.19.11` 将未显式配置时的默认 Approval Mode 从 `default`（未在 Allowlist 的 Tool Call 逐次询问）改为 `auto`。官方 PR 说明 Auto 使用三层决策：Workspace Edit 自动批准、只读 Tool 按 Allowlist 批准、其余操作交给 LLM Classifier；Untrusted Folder 仍强制回到 Ask Permissions，用户也可显式切回手动模式。PR 同时明确承认 Classifier 可能把风险操作误判为安全，且交互验证仅覆盖 macOS。稳定版 `v0.19.12` 又加入一项不可被 Auto/YOLO、Allow Rule、Permission Allow Hook 或同批自动审批满足的例外：主 Session 退出 Plan Mode 必须由 CLI、IDE、ACP 或 `stream-json` Host 返回明确决策，模式只在成功执行时原子切换；取消、中止、过期审批、并发退出或切换失败都保持在 Plan Mode。2026-07-19 合并的 PR #7172 进一步把 Plan Mode 的 Shell 与 Monitor 调用接入三态安全分类：只读命令沿用既有权限流，明确修改状态的命令在任何审批界面出现前即阻止，无法确定的命令只接受与 Plan 版本、工作目录、策略、原始请求和实际可执行调用绑定的一次性精确批准；持久、修改、过期、伪造或未在界面提供的批准均失败关闭。该契约贯穿 Core、ACP、`stream-json`、嵌套 Subagent、Teammate 与后台 Agent，但截至本次扫描尚未在稳定版 Release 中确认。

### 为什么重要

分析判断：默认值把安全控制重心从“每个动作由人确认”迁移到 Workspace Trust、路径边界、操作分类与事后验证。对团队而言，是否仍提供手动模式并不足够；需要检查默认模式能否被企业策略锁定、分类理由是否可审计，以及 Workspace 内写入为何被统一视为可自动批准。新增实现说明，审批安全还取决于“界面实际提供了什么选项”与“执行时收到的是否仍是同一次精确调用”能够跨 TUI、Host、后台队列和 Subagent 保持一致；否则外部 Host 或过期响应可能重新引入被界面隐藏的持久授权。

### 尚待确认

- LLM Classifier 的规则、模型、误报漏报率、超时或失败时是否关闭授权。
- Symlink、Submodule、Mount、生成式 Shell 命令与 Workspace Path Lock 组合时的实际边界。
- Windows/Linux 上的授权行为，以及组织是否能集中强制 Approval Mode。
- Plan Mode 的内在交互要求在公开云 ACP Host、Windows/Linux 及自定义 Host 中是否一致，失败通知与实际模式是否始终同步。
- PR #7172 尚未在稳定版 Release 中确认；真实模型驱动 TUI、外部托管 ACP Host、Windows 与 Linux 也未完成现场验证。
